While loop evalulates the condition first and executes the code second. while loop--> while(condition) { code inside }

Do While loop executes the code first and evalulates the condition second. Do first, then evaluate second. Guaranteed one execution before evaluating the condition. do while loop--> do { code inside } while(condition)

for loop--> for(initialization; condition; increment or decrement){ code inside }. No initialize needed--> for( ; condition; increment or decrement){ code inside }. Muliple variables can be initialized. for (x in something) { conde inside } also works.
